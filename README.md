# erdos-harness

A long-horizon multi-agent research harness applied to **Erdős Problem #872** (the antichain divisibility game). This repo is the working record of that research — round docs, prompts, formal verifications, phase probes, and the paper.

For the generic, clonable version of this harness (no Erdős content — just the operating manual, skills, templates, and scripts), see the companion `erdos-co-researcher` starter repo.

## If you want to read the Erdős 872 work

Start at `erdos-872/current_state.md`. Each round doc has YAML front-matter (`id`, `type`, `claim`, `action`, `failure_mechanism` where applicable) linking it to its dispatch prompt and to any round it refutes or extends. The compiled state view — generated by `scripts/compile_rounds.py` — is an indexed summary organized by Established / Ruled Out / Pending-target refutations / Indices.

The accompanying math + methodology papers will land on arXiv; see `docs/plans/` for the publishing roadmap.

## Core methodology

See `CLAUDE.md` for the full operating manual. Briefly:

- **Multi-role harness**: primary researcher (strong reasoning LLM), curator (Claude Code), informal verifiers (cross-family), formal verifier (Aristotle → Lean 4).
- **Factual-completeness doctrine**: researcher prompts are Problem / Established / Ruled Out / Numerical Evidence / Open Question — no hand-waved suggestions.
- **A/B/C branching dispatch**: varied framings across parallel threads beats identical-prompt sampling.
- **Cross-family convergence** as a truth signal; within-family attractor-basin mapping as a diagnostic.
- **Immutable round docs**: YAML front-matter with an `action` graph; state is derived, never mutated.
- **Commit per round**: the durability layer is git, not the filesystem.

Named failure modes surfaced during the Erdős 872 program (load-bearing for the methodology paper):

- **σ\*-trap** — hidden adversary-strategy assumption invalidating many rounds of upstream work
- **Within-family attractor basins** — same-family models converging on the same flaw
- **Static / dynamic bound conflation** — state inequalities mistaken for process bounds
- **Curator drift** — synthesis getting ahead of the evidence
- **Frontier-framing anti-pattern** — "this is a hard open problem" framing empirically correlates with 10–25 min reasoning traces vs 40–60 min on neutral framing
