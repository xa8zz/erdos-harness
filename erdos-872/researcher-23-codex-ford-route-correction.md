# Round 23 Codex — the old Ford-band rough-cofactor reduction is invalid as stated

I pushed directly on the best remaining upper-bound direction and found a genuine issue in the existing Ford-band reduction.

## Main point

The reduction in [researcher-18-codex-slow-growth-reduction.md](/Users/omisverycool/erdos-harness/erdos-872/researcher-18-codex-slow-growth-reduction.md:1) is **not valid as stated**.

It assumes that if every surviving upper move has a distinguished factorization
\[
u = a m,\qquad a\in(Y,2Y],\qquad P^-(m)>Y^\delta,
\]
then Ford 2019 implies
\[
\#\{\text{survivors in band }Y\}\ll \frac{n}{\log^2 Y}.
\]

But the imported Ford 2019 theorem counts
\[
H(x,y,2y;\mathcal R_w)
=
\#\{n\le x:\ P^-(n)>w,\ \exists d\mid n,\ y<d\le 2y\},
\]
that is: integers which are themselves `w`-rough.

The hypothesis above only says the **cofactor** `m` is rough. The skeleton `a` may have arbitrarily many small prime factors below `w`, so `u` need not lie in `\mathcal R_w`. Therefore Ford 2019 does not apply.

## Band-local counterexample to the naive reduction

This is not just a technical mismatch; the bandwise conclusion is false in that generality.

Fix any `\delta>0`, and choose any `\beta` with
\[
0<\beta<\frac{1}{1+\delta}.
\]
Set
\[
Y=n^\beta,\qquad C=\frac{Y}{30}.
\]

Let
\[
\mathcal C := \{c\in(C,2C] : c \text{ odd and squarefree}\},
\qquad
\mathcal A := \{30c : c\in\mathcal C\}\subseteq (Y,2Y].
\]

Then every `a\in\mathcal A` has a small prime factor `2`, so no `a` is `w`-rough for any `w\ge 2`.

Also
\[
\sum_{a\in\mathcal A}\frac1a

=
\frac1{30}\sum_{c\in\mathcal C}\frac1c
\gg 1,
\]
because the odd squarefree integers have positive density, so
\[
\#\mathcal C \asymp C,
\qquad
\sum_{c\in\mathcal C}\frac1c \asymp 1
\]
on a dyadic interval.

Now for each `a\in\mathcal A`, consider primes
\[
\ell\in\left(\frac{n}{2a},\frac{n}{a}\right].
\]
For such `\ell`,
\[
u=a\ell\in(n/2,n].
\]

Since `a\asymp Y=n^\beta` and `\beta(1+\delta)<1`,
\[
\ell>\frac{n}{2Y}>Y^\delta
\]
for large `n`. Hence each `u` has a distinguished factorization
\[
u=a m,\qquad a\in(Y,2Y],\qquad m=\ell,\qquad P^-(m)=\ell>Y^\delta.
\]

But the number of such upper-half integers is
\[
\sum_{a\in\mathcal A}\left(\pi\!\left(\frac{n}{a}\right)-\pi\!\left(\frac{n}{2a}\right)\right)
\gg
\frac{n}{\log n}\sum_{a\in\mathcal A}\frac1a
\gg
\frac{n}{\log n}.
\]

So in a **single dyadic band** we can have
\[
\gg \frac{n}{\log n}
\]
upper-half integers with a factorization
\[
u=a m,\qquad a\in(Y,2Y],\qquad P^-(m)>Y^\delta,
\]
while the claimed Ford-2019-style conclusion would predict only
\[
O\!\left(\frac{n}{\log^2 Y}\right)
=
O\!\left(\frac{n}{\log^2 n}\right).
\]

This contradicts the reduction by a full factor of `\log n`.

## Corrected lesson

The old Ford-band route used the wrong arithmetic class.

What Ford 2019 would legitimately count is a family of survivors satisfying the stronger condition
\[
P^-(u)>Y^\delta
\quad\text{and}\quad
\exists d\mid u\ \text{with}\ Y<d\le 2Y.
\]
In words:

- the **entire survivor** `u` must be `Y^\delta`-rough, not just the complementary factor;
- and `u` must have a divisor in the controlled band.

Under that stronger hypothesis, Ford 2019 can indeed yield bandwise bounds of size `n/\log^2 Y`.

## What survives of the Ford program

Two valid Ford-shaped directions remain:

1. **Corrected rough-whole-number route.**
   Prove that after setup, every unresolved upper move is itself `Y^\delta`-rough for some polynomial band `Y`, and has a divisor in `(Y,2Y]`.

2. **Band-local reciprocal-mass control.**
   If whole-number roughness is too strong, then one needs an additional theorem saying that in each dyadic band `(Y,2Y]`, the actual skeleton set `A_Y` generated by the safe-prime closure has sufficiently small reciprocal mass. Rough cofactor alone is not enough.

## Bottom line

The "rough complementary factor" version of the Ford-band reduction is false as a counting principle. Any viable upper-bound theorem must either:

- force the **whole survivor** into Ford's rough-divisor class, or
- add a separate bandwise control theorem on the multiplicative-closure skeletons.
