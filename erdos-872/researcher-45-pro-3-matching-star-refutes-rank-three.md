The weighted rank-three completion dichotomy is **false as stated** for arbitrary shared-coface defect-two clusters. The obstruction is not merely the earlier matching-star intuition; it can be written as a clean weighted local counterexample satisfying the defect-two density-increment hypothesis while avoiding every defect-three dense completion.

Here is the counterexample.

Fix

[
m=\lfloor \log h\rfloor,\qquad \theta = m^{-1},
]

and let

[
C={c_1,\ldots,c_{h-1}}
]

be a common defect-one coface. For each arm (i\in I), with (|I|\asymp h), define

[
D_i=C\setminus{c_i}.
]

Let (W_0) be a common certificate leaf set with

[
|W_0|=m.
]

For each (i), take a private packet-separated vertex set

[
V_i={x_{i,r},y_{i,r}:1\le r\le J},
\qquad
J\asymp m^2,
]

with all (V_i)’s mutually disjoint and disjoint from (W_0). Let

[
M_i=\bigl{{x_{i,r},y_{i,r}}:1\le r\le J\bigr}
]

be a matching on (V_i).

Now define the local rank-(h) section of (D_i) to consist of

[
{C\cup{w}:w\in W_0}
]

together with

[
{D_i\cup{x_{i,r},y_{i,r}}:1\le r\le J}.
]

All local coordinates may be chosen in the same dyadic expanded window, so their harmonic weights are comparable. Thus the weighted calculation is the same as the unweighted one up to absolute constants.

For each (D_i), the common coface certificate is

[
C(D_i)=C.
]

The certificate shadow has mass

[
\mu(\operatorname{Sh}(C))
\asymp m,
]

whereas

[
\mu(\operatorname{Sh}(D_i))
\asymp m+J
\asymp m^2.
]

Therefore

[
\mu(\operatorname{Sh}(C))
\asymp
\frac1m,
\mu(\operatorname{Sh}(D_i))
===========================

\theta,\mu(\operatorname{Sh}(D_i)).
]

So every (D_i) satisfies the defect-two density-increment condition at threshold

[
\theta\asymp\frac1{\log h}.
]

The cluster has large defect-two lifted union mass. Since (|I|\asymp h) and (J\asymp m^2),

[
\nu_{h,2}^{\cup}({D_i:i\in I})
\asymp
\frac{|I|J}{\binom h2}N_h
\asymp
\frac{m^2}{h}N_h.
]

The desired defect-two endpoint estimate would require

[
\ll \frac{N_h}{h}.
]

This construction is larger by a factor

[
m^2\asymp(\log h)^2.
]

Now test rank-three absorption.

Take the natural defect-three face associated to two arms,

[
S_{ij}=C\setminus{c_i,c_j}.
]

The section supplied by the cluster over (S_{ij}) consists of

[
{S_{ij}\cup{c_i,c_j,w}:w\in W_0},
]

plus

[
{S_{ij}\cup{c_j,x_{i,r},y_{i,r}}:1\le r\le J},
]

plus

[
{S_{ij}\cup{c_i,x_{j,r},y_{j,r}}:1\le r\le J}.
]

Hence its mass is

[
\asymp m+2J
\asymp m^2.
]

But the local coordinate support available to (S_{ij}) contains at least

[
{c_i,c_j}\cup W_0\cup V_i\cup V_j,
]

whose size is

[
\asymp J\asymp m^2.
]

The full local rank-three completion shadow over that support therefore has mass

[
\asymp J^3
\asymp m^6.
]

Thus the defect-three density of (S_{ij}) is at most

[
\frac{m^2}{m^6}
===============

# m^{-4}

o(\theta).
]

So (S_{ij}) is not dense at threshold (\theta\asymp 1/m).

Nor is this failure limited to the special faces (S_{ij}). Consider any defect-three lower face (S) contained in one of the constructed rank-(h) facets.

If (S\subset C), then (S=C\setminus{c_i,c_j}) for some (i,j), and the calculation above applies.

If (S) contains a common certificate leaf (w\in W_0), then (S) lies in at most one certificate facet (C\cup{w}), while its local ambient rank-three shadow still has many packet-separated completions. Its density is (O(J^{-1})) or smaller, hence (o(\theta)).

If (S) contains one private endpoint (x_{i,r}), then because the private endpoints form a matching, (S) is contained in at most one private matching facet (D_i\cup{x_{i,r},y_{i,r}}). Again its local ambient shadow has at least polynomially many packet-separated completions, so its density is (o(\theta)).

If (S) contains two private endpoints, it is contained in at most one matching facet, and the same bound applies.

Therefore no defect-three face generated by the cluster has residual local density (\gg\theta).

This refutes the desired weighted rank-three completion dichotomy in the following precise form:

[
\boxed{
\text{Large shared-coface defect-two mass does not force dense defect-three completion.}
}
]

The obstruction is sparse support entropy. The defect-two residue graphs (G_i=M_i) have large edge mass,

[
|G_i|\asymp m^2,
]

but they are matchings on support size

[
|V_i|\asymp m^2.
]

A rank-three completion argument needs the residue mass to live on small or high-degree support. In the pure star, one has

[
G_i=\binom{W}{2},
\qquad |W|\asymp m,
]

so the two-leaf residue is dense and rank-three absorption works. In the matching-star, one has the same edge mass but spread over (\asymp m^2) private vertices, so every rank-three section is sparse.

In weighted notation, if (G_i) has vertex mass

[
\sigma_i=\sum_{v\in \operatorname{supp}G_i}\omega_v
]

and edge mass

[
B_i=\sum_{{u,v}\in G_i}\omega_u\omega_v,
]

then a pair (i,j) can only give rank-three density on the order of

[
\delta_{ij}
\lesssim
\frac{A+B_i+B_j}{(A+\sigma_i+\sigma_j)^3},
]

where (A=\mu(\operatorname{Sh}(C))). The defect-two density increment gives

[
B_i\asymp \theta^{-1}A,
]

but it gives no upper bound on (\sigma_i). In the matching construction,

[
\sigma_i\asymp B_i,
]

so

[
\delta_{ij}
\asymp
B_i^{-2}
\asymp
m^{-4}
\ll
m^{-1}
======

\theta.
]

Thus the attempted proof fails for a structural reason: edge mass alone does not force rank-three density. One also needs an effective-support or weighted-degree concentration hypothesis.

The correct replacement cannot be a rank-three completion dichotomy. It must be at least a trichotomy:

[
\boxed{
\textbf{Shared-coface defect-two trichotomy}
}
]

For a high-multiplicity shared-coface cluster, either

[
\nu_{h,2}^{\cup}(\mathcal F)\ll \frac{N_h}{h},
]

or there is a genuine dense defect-three completion, or a positive proportion of the mass is carried by sparse private matching residues satisfying something like

[
\Delta_w(G_i)
:=
\max_v\sum_{u:{u,v}\in G_i}\omega_u\omega_v
\ll
B_i
]

and

[
\sigma_i^3\gg \theta^{-1}B_i.
]

The third alternative is exactly the matching-star obstruction.

So the newly isolated missing ingredient is not a weighted rank-three completion theorem. It is a **sparse matching-residue packing theorem**, for example:

[
\boxed{
\sum_{\text{sparse matching-residue clusters}}
\nu_{h,2}^{\cup}(\mathcal F)
\ll
\frac{N_h}{h}.
}
\tag{SMRP}
]

For the conjectured sharp rate, the needed version would be much stronger:

[
\boxed{
\sum_{\text{sparse matching-residue clusters}}
\nu_{h,2}^{\cup}(\mathcal F)
\ll
N_h,h\log h,e^{-h}.
}
\tag{sharp SMRP}
]

Without such a sparse-residue packing theorem, the defect-two endpoint remains open. The matching-star construction shows that pure star absorption and rank-three dense completion are insufficient, even in the weighted lifted harmonic framework.
