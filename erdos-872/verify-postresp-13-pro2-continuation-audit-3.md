# Audit #3 of Pro-2 continuation theorem (static post-hoc obstruction on case (b))

This is a rigorous, adversarial audit of the proposed static obstruction theorem. The mathematical core of the response holds up under pressure, but it contains a critical strategic vulnerability regarding Prolonger's move budget that requires a sparse-subset optimization to fix. Furthermore, its scope against the actual Resolution Theorem is narrower than it implies. 

Here is the line-by-line breakdown of the pressure points.

### 1. Circuit-Lemma Application: Airtight
The application of the Circuit Lemma to the support $T = \{q, a, b\}$ is structurally flawless. 
* **(a) Incomparability:** Every carrier in $\mathcal{E}_Q$ explicitly requires the prime $2$. Because $m = qab$ is the product of odd primes, $2 \notin T$. A set containing $2$ can never be a subset of a set lacking $2$, nor vice versa. Therefore, $T$ is strictly incomparable with every carrier. It is irrelevant if a prime $q \in Q$ accidentally coincides with the label "$b$" in another carrier; the presence of $2$ acts as an absolute barrier to comparability.
* **(b) Proper subsets:** The checks $\{q, a\} \subseteq \{2, q, a\}$ and $\{q, b\} \subseteq \{2, q, b\}$ are sound because $2qa \le n$ and $2qb \le n$ are enforced by $m \le n$ (since $b \ge 3$, $qa \le n/3 \Rightarrow 2qa \le 2n/3 \le n$). 

### 2. Proper-Multiples Argument: Saved by the Open Boundary
The claim that no proper multiple of $m$ is legal survives solely due to the strict inequality $m > n/3$. 
* If $m$ were allowed to equal $n/3$, then $3m$ could equal $n$. The support of $3m$ would be $\{3, q, a, b\}$. Because every carrier in $\mathcal{E}_Q$ contains $2$, $\{3, q, a, b\}$ would **not** contain any carrier support, making $3m$ a legal proper multiple and destroying the "isolated maximal" claim. 
* However, because the domain is strictly $(n/3, n]$, $3m > n$ unconditionally. The only valid multiple $\le n$ is $2m$, which trivially contains the carrier $2qa$. The boundary logic is perfectly sharp.

### 3. The $|\mathcal{M}_Q|$ Counting: Asymptotically Robust
The analytic number theory bounds are applied correctly.
* Mertens' sum over $Q = [n^\beta, n^\gamma]$ yields $\log(\gamma/\beta) \asymp 1$.
* The sum over $a \le n^\delta$ yields $\sim \log\log n$.
* The Prime Number Theorem bound for $b \in (X/3, X]$ holds because $X \ge n^{1-\gamma-\delta}$, which goes to infinity as long as $\gamma + \delta < 1$.
* **Double-counting:** The map from unordered triples to $m$ is unique by the fundamental theorem of arithmetic. While the bounds on $q, a, b$ might overlap (allowing a single $m$ to be generated by different permutations of roles), the multiplicity of overcounting is strictly bounded by $3! = 6$. A constant factor overcount cannot break a $\gg$ asymptotic bound. The scaling holds.

### 4. Post-Hoc Scope Claim: Attacking a Strawman
The response claims this rules out "any extension of the Resolution Theorem... that waits for the carrier prefix to be installed." This is technically true, but strategically weak.
* **(a) The Real RT is Interleaved:** The classical Resolution Theorem (and the rank-3 extension) is **not** post-hoc. Shortener's Phase 1 plays *every legal prime* during or before Prolonger builds the composite structure. By playing small primes early (e.g., the prime $2$), Shortener instantly invalidates the $2ab$ and $2qa$ carriers required to form $\mathcal{E}_Q$. 
* **(b) Scope Constraint:** The theorem successfully proves that Shortener *must* play an interleaved or online strategy (like Phase 1 prime-rushing), but since Shortener already does exactly this in the known $O(n/\log n)$ proofs, the obstruction does not threaten the established paradigms.

### 5. Budget Self-Reference: The Fatal Flaw & The Sparse Rescue
This is the most vulnerable point in the response. As defined, the family $\mathcal{E}_Q$ contains *all* $2ab \le n$. The number of such pairs is $\Theta(n \log\log n / \log n)$. 
* **The Tautology:** If Prolonger must spend $\Theta(n \log\log n / \log n)$ moves to install the prefix, the theorem merely proves that spending $N$ moves forces Shortener to spend $N$ moves. It fails to establish $L(n) \ge n \log\log n / \log n$ unconditionally because it assumes Prolonger has the budget to build $\mathcal{E}_Q$ under an $O(n/\log n)$ constraint—a logical contradiction.
* **The Sparse Rescue:** The theorem can be saved, but the response failed to do the math. Prolonger does not need *all* $2ab \le n$; they only need the carriers for the specific $\{a, b\}$ pairs that appear in $\mathcal{M}_Q$. 
    * For a fixed $a \le n^\delta$, the required $b$'s fall into the union of intervals $\bigcup_{q \in Q} (n/(3qa), n/(qa)]$, which collapses to $(n/(3n^\gamma a), n/(n^\beta a)]$.
    * The number of distinct primes $b$ in this span is roughly $\pi(n^{1-\beta}/a)$. 
    * Summing over $a$, the number of distinct $\{a, b\}$ pairs is $O(n^{1-\beta} \log\log n / \log n)$. 
    * Because $\beta > 0$, $n^{1-\beta}$ is strictly sublinear. Thus, the required $\{2ab\}$ and $\{2qa\}$ carriers number $o(n/\log n)$. Only the $\{2qb\}$ carriers cost $O(n/\log n)$. 
    * **Conclusion:** Prolonger *can* isolate $\gg n \log\log n / \log n$ moves while only spending $O(n/\log n)$ carriers. The response stumbled into a tautological definition of $\mathcal{E}_Q$, but a sparse optimization genuinely rescues the obstruction.

### Final Audit Verdict
The theorem is mathematically highly impressive but flawed in its exposition. It successfully proves a static lower bound against post-hoc resolution. However, the author missed the critical "sparse subset" budget optimization (Point 5), without which the theorem is a tautology. Furthermore, because it attacks a "wait-and-see" Shortener, it does not rupture the actively interleaved $O(n/\log n)$ proofs currently in play.
