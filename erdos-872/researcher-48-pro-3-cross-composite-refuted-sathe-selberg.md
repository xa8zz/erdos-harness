The named **Cross-Activator Composite Quotient Budget Lemma is false** in the operational form needed for the Carleson argument.

A single (\sigma^\star)-forced legal cross-composite can first-touch a positive proportion of the central rank-(h) harmonic mass. That is already much larger than

[
\frac{h^2}{\log n}N_h,
]

since (h\sim \log\log n).

I will state the refutation in the squarefree rank-(h) setting, which is the harder version. If the degree uses all (\Omega=h) integers, the same construction only becomes stronger.

Let

[
\mathcal R_h(n)={m\in(n/2,n]:m\text{ squarefree},\ \omega(m)=h},
\qquad
N_h=\sum_{m\in\mathcal R_h(n)}\frac1m.
]

For a current chosen antichain (A), define

[
\deg_A(x)=
\sum_{\substack{m\in\mathcal R_h(n)\ x\mid m\ m\text{ unresolved by }A}}\frac1m.
]

The Shortener strategy (\sigma^\star) chooses a legal (x) maximizing (\deg_A(x)).

Take

[
h=h(n)\sim \log\log n.
]

The Prolonger strategy is the fixed five-move prefix

[
21,\quad 35,\quad 91,\quad 133,\quad 203,
]

i.e.

[
7\cdot 3,\quad 7\cdot 5,\quad 7\cdot 13,\quad 7\cdot 19,\quad 7\cdot 29.
]

Against (\sigma^\star), the Shortener replies are, for all sufficiently large (n),

[
2,\quad 11,\quad 17,\quad 23,\quad 15.
]

Thus the actual legal play prefix is

[
21,\ 2,\ 35,\ 11,\ 91,\ 17,\ 133,\ 23,\ 203,\ 15.
]

The final Shortener move (15=3\cdot5) is a legal cross-activator composite: (3) is individually illegal because (3\mid21), and (5) is individually illegal because (5\mid35), but

[
15\nmid21,\quad 21\nmid15,\quad 15\nmid35,\quad 35\nmid15,
]

and similarly (15) is incomparable with (91,133,203,2,11,17,23).

The finite legality check is immediate: every number in the displayed sequence is incomparable with every earlier number.

The central-rank degree computation uses the fixed-prime local Sathe law. Under harmonic measure on (\mathcal R_h(n)), for every fixed finite set of primes, the divisibility indicators converge jointly to independent Bernoulli variables with

[
\Pr(p\mid m)\to \frac1{p+1},
\qquad
\Pr(p\nmid m)\to \frac p{p+1},
]

because (h/\log\log n\to1). This is the squarefree Euler-factor form of Sathe-Selberg: for fixed marked primes (P_0),

[
\sum_{m\in\mathcal R_h(n)}\frac1m\prod_{p\in P_0} y_p^{1_{p\mid m}}
===================================================================

N_h
\prod_{p\in P_0}\frac{p+y_p}{p+1}
+o(N_h).
]

Now write

[
\alpha_p=\frac1{p+1},
\qquad
\bar\alpha_p=\frac p{p+1}.
]

After Prolonger has played (7q) for (q\in Q), a legal prime (p\notin Q\cup{7}) has limiting unresolved degree

[
D_p(Q,C)
========

\alpha_p
\left(
\bar\alpha_7+\alpha_7\prod_{q\in Q}\bar\alpha_q
\right)
\prod_{c\in C}\bar\alpha_c,
]

where (C) is the set of already claimed Shortener primes. The parenthesized term says: either (7\nmid m), or (7\mid m) but none of the protected (q)’s divides (m), since (7q\mid m) would make (m) already resolved.

For a legal cross-composite (T\subseteq Q), (|T|\ge2), the limiting unresolved degree is

[
D_T(Q,C)
========

\left(\prod_{q\in T}\alpha_q\right)
\bar\alpha_7
\prod_{c\in C}\bar\alpha_c.
]

Among all legal cross-composites generated by (Q), the largest one is always (T={3,5}), namely (15), because it uses the two smallest protected primes.

The exact comparisons are:

[
\frac{D_{15}}{D_{11}}=\frac{28}{61}<1,
]

[
\frac{D_{15}}{D_{17}}=\frac{196}{283}<1,
]

[
\frac{D_{15}}{D_{23}}=\frac{3136}{3383}<1,
]

but after Prolonger plays (203=7\cdot29),

[
\frac{D_{15}}{D_{31}}
=====================

\frac{125440}{101243}>1.
]

Therefore (\sigma^\star) is forced to play

[
2,\ 11,\ 17,\ 23,\ 15
]

in that order, for all sufficiently large (n). Legal composites containing a still-legal prime are dominated by that prime’s degree; legal cross-composites not containing a legal prime are products of protected primes from (Q), and (15) is the largest of those. So no omitted legal integer can beat the displayed (\sigma^\star) move.

At the final turn, the newly removed rank-(h) mass from the legal cross-composite (15) is

[
u_5
===

\deg_A(15),
]

where (A={21,2,35,11,91,17,133,23,203}). Its limiting mass is

[
\frac{u_5}{N_h}
\to
\alpha_3\alpha_5\bar\alpha_7\bar\alpha_2\bar\alpha_{11}\bar\alpha_{17}\bar\alpha_{23}.
]

Explicitly,

[
\alpha_3\alpha_5\bar\alpha_7\bar\alpha_2\bar\alpha_{11}\bar\alpha_{17}\bar\alpha_{23}
=====================================================================================

# \frac{30107}{1492992}

0.0201655467\ldots
]

So for all sufficiently large (n),

[
u_5\ge 0.01,N_h.
]

But the proposed cross-composite budget would require, for some absolute (C),

[
Q_h^\times
\le
C,\frac{h^2}{\log n}N_h.
]

Any operational (Q_h^\times) capable of controlling first valid (\sigma^\star)-certificates must count this first-touch cross-composite mass, hence

[
Q_h^\times\ge u_5\ge 0.01,N_h.
]

Since (h\sim\log\log n),

[
\frac{h^2}{\log n}\to0.
]

Thus for every fixed (C),

[
C,\frac{h^2}{\log n}N_h
<
0.01,N_h
]

for all sufficiently large (n), contradicting the claimed budget.

So the missing lemma is not merely unproved; it is false.

The failure mechanism is very specific: small protected primes have positive central-rank harmonic density. Once Prolonger has protected (3) and (5) on different activators, and then waits while (\sigma^\star) legally claims (2,11,17,23), the composite (15) becomes the highest-degree legal move. It first-touches a constant fraction of (N_h), not (O(h^2N_h/\log n)).

This refutation does not rely on burst-star multiplicity or sparse matching-star residues. The budget already fails in a one-certificate cross-composite configuration.

Here is the exact sandbox-verifiable computation of the finite constants and legality.

```python
from fractions import Fraction
from itertools import combinations

def alpha(p):
    return Fraction(1, p + 1)

def beta(p):
    return Fraction(p, p + 1)

def H(Q):
    # no chosen activator 7*q divides the rank-h facet:
    # either 7 is absent, or 7 is present and every q in Q is absent
    prod = Fraction(1, 1)
    for q in Q:
        prod *= beta(q)
    return beta(7) + alpha(7) * prod

def D_prime(p, Q, C):
    val = alpha(p) * H(Q)
    for c in C:
        val *= beta(c)
    return val

def D_cross(T, C):
    val = beta(7)
    for q in T:
        val *= alpha(q)
    for c in C:
        val *= beta(c)
    return val

states = [
    ("after P=35",  [3, 5],             [2],             11),
    ("after P=91",  [3, 5, 13],         [2, 11],         17),
    ("after P=133", [3, 5, 13, 19],     [2, 11, 17],     23),
    ("after P=203", [3, 5, 13, 19, 29], [2, 11, 17, 23], 31),
]

for name, Q, C, p in states:
    d15 = D_cross((3, 5), C)
    dp = D_prime(p, Q, C)
    print(name)
    print("  D15/Dprime =", d15 / dp, "=", float(d15 / dp))

final_mass = D_cross((3, 5), [2, 11, 17, 23])
print("final u5/N_h =", final_mass, "=", float(final_mass))

def legal(x, chosen):
    return x >= 2 and all(x != y and x % y != 0 and y % x != 0 for y in chosen)

sequence = [
    ("P", 21),
    ("S", 2),
    ("P", 35),
    ("S", 11),
    ("P", 91),
    ("S", 17),
    ("P", 133),
    ("S", 23),
    ("P", 203),
    ("S", 15),
]

chosen = []
for player, x in sequence:
    assert legal(x, chosen), (player, x, chosen)
    chosen.append(x)

print("all moves legal")
print("3 legal before final S?", legal(3, chosen[:-1]))
print("5 legal before final S?", legal(5, chosen[:-1]))
print("15 legal before final S?", legal(15, chosen[:-1]))
```
