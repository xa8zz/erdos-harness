**Real Gaps**

There are really four theorem-sized gaps now.

- **Gap 1: audit T2 itself.** The candidate `Ω(n(\log\log n)^2/\log n)` lower story still matters enormously. If it has a bug, the whole target changes. If it survives, any upper-bound route has to match that scale somehow.
- **Gap 2: control multiplicative closure, not just pairs.** The Pro counterexample says pair-shadow mass only controls the 2-layer. It does not stop a “safe” small-prime set from generating a huge squarefree closure and then an `Ω(n)` upper-half family. Any repaired round-20 theorem has to bound the closure generated by safe small primes, not just `Σ_2(P)`.
- **Gap 3: robust residual width for a sublinear certificate family.** The reduction in [researcher-18-pro-matching-sublinear-reduction-lemma.md](/Users/omisverycool/erdos-harness/erdos-872/researcher-18-pro-matching-sublinear-reduction-lemma.md:1) says we need a family `C_n=o(n)` such that once all of `C_n` are made illegal, the residual antichain width is also `o(n)`. The natural candidate is something like `C_{n,h}={Ω≤h}` with slowly growing `h`, but no one has a residual-width theorem for it.
- **Gap 4: a genuine within-fiber theorem.** The linear route only comes back if someone proves a new positive-density capture theorem inside one `b`-fiber. The old fan/shadow architecture cannot do it.

**Best Angles**

I’d rank the live angles like this:

1. **Hard re-audit the T2 lower bound.** This is the highest-leverage task. If T2 weakens, the pair-shadow collapse matters less and `n/log n` becomes much more plausible. If T2 survives, we know we need a stronger upper theorem than anything we currently have.

2. **Reformulate the dead 2-shadow route around closure.** The right object is no longer “which semiprime pairs got activated?” It is more like:
   \[
   S_t=\{\text{small primes still usable by Prolonger at time }t\},
   \]
   together with the squarefree closure generated by `S_t` up to `n/(2y)`. A viable theorem would have to show that under a concrete Shortener order, this closure never supports more than sublinear upper-half width.

3. **Push the certificate-family route.** This still feels like the most abstractly correct sublinear framework. We do not need to solve the full game directly; we need a small certificate family with a strong robust residual-width theorem. That is exactly what [researcher-18-pro-matching-sublinear-reduction-lemma.md](/Users/omisverycool/erdos-harness/erdos-872/researcher-18-pro-matching-sublinear-reduction-lemma.md:1) isolates.

4. **Push the Ford-band route.** If Shortener can force every survivor into factorizations
   \[
   u=a\,m,\quad a\in(Y,2Y],\quad P^-(m)>Y^\delta,\quad Y\ge n^{\gamma(n)},
   \]
   with `δ>0.278...`, then Ford-style counting gives a slow-growth upper bound. This is still the cleanest “top-down” sublinear program in [researcher-18-codex-slow-growth-reduction.md](/Users/omisverycool/erdos-harness/erdos-872/researcher-18-codex-slow-growth-reduction.md:1).

5. **Only then revisit linear lower bounds.** Right now, linear looks like it needs something qualitatively new, not a refinement of the existing fan machinery.

My current synthesis is: the research is now bottlenecked less by clever counting and more by choosing the right state variable. Pair-shadow was too shallow; the next successful invariant probably has to see either **squarefree closure** or **residual width under a certificate family**.
