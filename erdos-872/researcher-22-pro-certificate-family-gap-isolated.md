# Round 22 Pro — certificate-family residual-width gap precisely isolated

## Context

Follow-up to the Pro dispatch that produced the dynamic multiplicative-closure reduction theorem $L(n) \le 2|C_n| + 1 + \sup_{\text{reachable}} w(R_{C_n}(P, S))$. The user asked Pro to continue by producing a specific sublinear $C_n$ with sublinear residual width, which would give $L(n) = o(n)$. Pro honestly concluded that this step is exactly the unresolved gap in the program, identified the precise missing lemma, analyzed the natural smooth-squarefree candidate $C_y$, and showed via an entropy computation that the crude divisor-incidence approach fails quantitatively.

## Pro's response

I cannot honestly produce the requested (C_n) and proof from the supplied material. That step is exactly the unresolved “robust residual-width for a sublinear certificate family” gap.

The reduction I gave is valid as a target:

[
L(n)\le 2|C_n|+1+\sup_{\text{reachable }(P,S)}
w!\left(R_{C_n}(P,S)\right),
]

but I do **not** have a rigorous proof that any explicit sublinear (C_n) makes the supremum (o(n)). Giving one would prove (L(n)=o(n)), which is stronger than every established upper bound in the prompt.

The most natural candidate is

[
C_y:={d\le n/2:\ d\text{ squarefree and }P^+(d)\le y},
]

with (y=\exp((\log n)^{1/2})) or similar. This is sublinear, since

[
|C_y|\le \Psi(n,y)
==================

n\exp!\left(-(1+o(1))u\log u\right),
\qquad
u=\frac{\log n}{\log y}\to\infty.
]

If Shortener could sweep (C_y), then every later legal (x) whose (y)-smooth squarefree core

[
\operatorname{rad}*y(x):=\prod*{\substack{p\mid x\p\le y}}p
]

lies in (C_y) would have to satisfy

[
\operatorname{rad}*y(x)\in \operatorname{Cap}*{C_y}(P).
]

So the residual would be controlled by the down-closed complex

[
\mathcal K(P)
:=
{d\in C_y:\ d\mid p\text{ for some Prolonger sweep move }p}.
]

A sufficient estimate would be

[
\sum_{d\in \mathcal K(P)}\frac1{\varphi(d)}
=o(\log y)
]

uniformly over reachable (P), because then the number of possible residual integers would be

[
\ll
\frac{n}{\log y}
\sum_{d\in \mathcal K(P)}\frac1{\varphi(d)}
+o(n)
=o(n).
]

The problem is that this is precisely where the proof breaks. The reachable complex (\mathcal K(P)) is not controlled merely by its pair-shadow, by the number of Prolonger moves, or by static smooth-number estimates. Prolonger can build overlapping facets of the squarefree smooth complex, and the known two-shadow counterexample shows that pair-level control is too weak. Extending the control to all ranks is the missing multiplicative-closure theorem.

A tempting incidence argument also fails quantitatively. Suppose one tries a central-divisor certificate family such as

[
C_k:={d\le n/2:\ d\text{ squarefree},\ \omega(d)=k},
\qquad k\sim \theta\log\log n.
]

For a typical (x) with (\omega(x)\sim H:=\log\log n), the number of (C_k)-divisors is roughly

[
\binom{H}{\theta H}
===================

\exp!\left(H,\mathsf H(\theta)+o(H)\right),
]

where (\mathsf H) is binary entropy. But the reciprocal mass of (C_k) is bounded by

[
\sum_{d\in C_k}\frac1d
\le
\frac1{k!}\left(\sum_{p\le n}\frac1p\right)^k
=============================================

\exp!\left(\theta H(1-\log\theta)+o(H)\right).
]

The exponent comparison is unfavorable:

[
\theta(1-\log\theta)-\mathsf H(\theta)
======================================

\theta+(1-\theta)\log(1-\theta)>0
]

for (0<\theta<1). Thus the crude divisor-incidence method gives no (o(n)) residual bound.

So the state is:

[
\boxed{
\text{No explicit sublinear }C_n\text{ with }
\sup_{\text{reachable}}w(R_{C_n})=o(n)
\text{ is presently proved from the supplied framework.}
}
]

The exact missing lemma would be something like:

[
\boxed{
\text{For some }y\to\infty,\ y=n^{o(1)},\
\sup_{\text{reachable }P}
\sum_{d\in \mathcal K(P)}\frac1{\varphi(d)}
=o(\log y).
}
]

That would immediately give (L(n)=o(n)) through the closure theorem. But proving it requires new control of the full squarefree multiplicative closure generated by Prolonger’s sweep moves, not just primes, semiprimes, or any fixed-rank shadow.

So the definitive continuation is negative in the following precise sense: the reduction identifies the right object, but the requested sublinear certificate family plus (o(n)) residual-width proof is not available from the current theorem stack. Any claimed proof at this point would need to supply the missing full-rank closure estimate above.
